# Aurora Tunes

A modern music streaming web application powered by **Spotify API + Firebase**, featuring search, playlists, user profiles, and a fully interactive audio player.

---

## Features

- Email/Password & Google Authentication (Firebase)
- Spotify-based Search (Tracks, Artists, Albums)
- 30-sec Song Preview Playback
- Artist & Album Pages
- Playlist Creation & Management
- Add/Remove Songs from Playlists
- User Profile System (Edit + Upload Photo)
- Dashboard (Trending, New Releases, Genres)
- Shuffle & Repeat Modes
- Volume Control & Player State Persistence
- Firestore Database Integration
